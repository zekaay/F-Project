About

"Ecommerce Genius" is a complete solution for Multi Vendor Ecommerce Business Management System. This CMS Includes almost everything you need to make a ecommerce business website. Paypal, Stripe and Cash of Delivery is integrated for easy payment transaction. It has many important section which are important for any ecommerce business website and everything is fully dynamic. This script is fully responsive for any device. Admin can manage complete website without single line of coding knowledge. It has strong SQL injection protection system which will keep away this system from hackers.

Installation

Download the file from codecanyon then extract in your computer and you will get three folders.

Step 1 : Extract files.zip file from Files folder to server.

Step 2 : Create database using database.sql from Database folder.

Step 3 : Edit details of .env file in project folder in server.

Dashboard

The dashboard is very simple, informative and dynamic. Admin can change his account details from top right corner. So admin doesn't need to do this manually from database.

Orders

All the orders will be here. Admin can Mark Complete any order and contact with the client.

Products

All the Products will be here. Admin can add, edit or delete from there. Featured image size must be 600x750px or same ratio.

Customers

All the customers details will be here. Admin can contact with the client.

Manage Category

There will be all the categories. If you want to delete any category or subcategory or child category remember that all the products under those will be deleted. So be careful to delete any of these. for slugs, just use small letters and no special character or space. you can use only small letter with hyphen, because it will be used for url. You can feature any category or subcategory to show on the homepage with featured image. Image must be square size.

Blog

You can add blog post here to show on the homepage. There should be 4 posts to show it perfectly.

Slider Settings

Admin Can add Slider image, title and slider text from here. Slider text can be decorated as Righ, left of middle position.

Language Settings

Admin can add or edit language from this section. Admin Just need to add new text in his/her language.

Testimonial Section

Admin can add or edit from this section. Title and Title bottom text can be added from here. Everything is dynamic in this system. Admin must add atleast 3 testimonials to run this section correctly.

Theme Color Settings

Admin can select a color then save it. The website theme color will be changed immediately. If you can't see new color after saving just use another browser to check or delete the browser cash and check the frontend.

Page Settings

You can Enable or Disable About Us, FAQ and Contact us Page. Just need a single click. You can add multiple email for the Contact Us page email address. Just separate emails using comma and don't give any space. You can add brand logos large banner and home banners for your website from here to show on homepage.

Social Settings

This script allows you to add social links in your website very easily. You just need to put the link. If you want to disable any social media then you can do it by single click and it will not appear in the Website. You can Enable it again by single click.

SEO Tools

Google Analytics

This script offers you easily Google Analytics integration system, it is a freemium web analytics service offered by Google that tracks and reports website traffic. You just need to put your Google Analytics script in the form and everything will be done.

Meta Keywords

You can add meta keywords in your website by writing them in this form. Every keyword must be separated using comma. It will help you for better SEO.

N.B. Please don't use any space after comma.

General Settings

You can add logo, favicon, footer details, background image and pickup locations from here.
In website content you can enable or disable any section.
In Sayment Settings you can set your payment options.

Subscribers

You can see all the subscribers of your website in the Subscribers section. You can download the list by single click in .csv format.

Multi Vendor

After registering as multi vendor, admin will get your details and approve you. Then vendor can add his/her products to sell. When any order will be placed then the vendor will get and order in his dashboard as well as admin. When vendor will mark any order complete then admin will get the status and after admin mark that order complete the vendor will be paid. When the vendor will do a withdraw request then it will go to the admin. Admin will take a small profit from the vendor.
